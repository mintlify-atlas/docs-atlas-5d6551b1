<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold dark:text-white text-gray-900 mb-4">Quick Start</h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">Get your e-commerce platform running in minutes</p>
  
  <Steps>
    <Step title="Clone the repository">
      Start by cloning the Don Palito Jr repository to your local machine.
      
      ```bash
      git clone https://github.com/DarkerJB/donpalitojrweb.git
      cd donpalitojrweb
      ```
    </Step>
    
    <Step title="Install dependencies">
      Install all required dependencies for the backend, admin panel, and web application.
      
      ```bash
      npm install
      ```
      
      <Note>This command will install dependencies for all three main components: backend, admin, and web.</Note>
    </Step>
    
    <Step title="Configure environment variables">
      Set up your environment variables for authentication, database, and payment processing.
      
      ```bash
      # Backend .env
      MONGODB_URI=your_mongodb_connection_string
      CLERK_PUBLISHABLE_KEY=your_clerk_key
      STRIPE_SECRET_KEY=your_stripe_key
      CLOUDINARY_CLOUD_NAME=your_cloudinary_name
      ```
      
      <Tip>See the [Environment Variables](/deployment/environment-variables) page for a complete list of required variables.</Tip>
    </Step>
    
    <Step title="Start the development servers">
      Launch the backend API and frontend applications.
      
      ```bash
      # Start backend
      npm run dev --prefix backend
      
      # Start admin panel (in another terminal)
      npm run dev --prefix admin
      
      # Start web app (in another terminal)
      npm run dev --prefix web
      ```
      
      Your applications will be available at:
      - Backend API: `http://localhost:3000`
      - Admin Panel: `http://localhost:5173`
      - Web App: `http://localhost:5174`
    </Step>
  </Steps>
</div>
